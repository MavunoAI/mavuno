ADR-0007 — Foundation Must Be Pure Dart

Decision:

packages/foundation will be a pure Dart package.

That means:

❌ No Flutter dependency
❌ No Android dependency
❌ No FFI
❌ No platform channels

Only:

Dart SDK
Immutable types
Value objects
Result types
Exceptions
Diagnostics
Time abstractions

This ensures the package can be reused by:

Mavuno CLI
Android application
Desktop application
Server-side tools (if ever needed)
Unit tests
Future benchmark tooling
