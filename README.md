# mavuno
Offline-first multilingual Edge AI platform for African agriculture.

## Vision

Empowering African farmers with AI that works without Internet.

## Features

- Offline AI
- African languages
- Weather prediction
- Flood prediction
- Crop advisory
- Livestock advisory
- Market intelligence
- Computer Vision
- Explainable AI

## Platforms

- Android
- Windows
- Linux
- macOS
- Edge Devices
